# OZ Browser

OZ Browser is a lightweight, customizable web search interface with dynamic background control. Users can search Google directly, browse a slider of preset wallpapers, and upload their own images to instantly set as the background.

## Features

- **Google Search**: Type a query in the input box and press Enter to search Google.
- **Dynamic Background**: Click any slider image to instantly set it as the page background.
- **Custom Uploads**: Upload an image from your desktop to immediately change the background.
- **Image Slider**: Browse preset and uploaded images with left/right arrow buttons.
- **Toggle Slider Visibility**: Show or hide the slider with a toggle button.
- **Smooth Transitions**: Background changes fade smoothly for a clean visual effect.
